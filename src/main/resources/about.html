ApkToolBoxGUI {VERSION}

APKToolBoxGUI is a handy tool for programmer with user-friendly GUI. It's a collection of common tools, encoding convert, timestamp convert, color picker and so on, furthermore there are some specialized tools for Android development(That is why it is named ApkToolBoxGUI). For details, please visit: https://github.com/jiangxincode/ApkToolBoxGUI
